- #3 Adicione um botão que permita ordenar os movimentos em ordem crescente ou decrescente. :heavy_check_mark:

- #4 Quando alguém ganha, destaque os três quadrados que causaram a vitória (e quando ninguém ganha, exiba uma mensagem informando que o resultado foi um empate). (in progress)
